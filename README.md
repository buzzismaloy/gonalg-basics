# Go Fundamentals Course Notes (Yandex)

This repository contains my structured notes and practice materials from the [“Go Fundamentals” course by Yandex Practicum](https://start.practicum.yandex/go-basics/).

The project is organized by sprints, topics, and lessons, reflecting the original course structure.

## Purpose of This Repository

- Document my learning progress in Go
- Build a personal knowledge base and reference
- Store structured notes and practice solutions
- Prepare for technical interviews and real-world Go development

## How to Use

1. Navigate through sprint folders
2. Open topic and lesson directories
3. Review notes and source code
4. Run examples locally:

```bash
go run main.go
```
